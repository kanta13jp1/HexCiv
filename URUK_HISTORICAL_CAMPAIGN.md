# ウルク史実キャンペーン — 設計確定事項と第2基盤

## 目的

既存のランダム4Xを残したまま、史実の位置・年代・成立条件を持つ別モードを追加する。
第1完成版は紀元前4000～3000年の南メソポタミアを扱う「ウルク—都市の夜明け」。
約50ターンの縦切り版で、都市化・河川農業・身分人口・実在物資・交易を段階的に遊べるようにする。

「歴史上のすべて」は有限の完了宣言ではなく、出典付き安定IDを後方追加する継続台帳として扱う。

## `/grill-me` で確定した製品判断

- 核はCivilization型の1人用4X。1ヘックス1部隊で世界地図上のターン制戦闘。
- 大量の歴史要素は、それぞれ固有効果を持つゲーム要素にする。
- ゲームバランスより史実再現を優先する。
- 文明・地域・最速登場年代を原則固定し、時代尚早の研究は不可。
- 世界地図上で国家が史実年代・史実地域に成立する。結果は固定せず、地理・人口移動・外交圧力を条件として再現する。
- プレイヤーは国家を操作し、国家滅亡で敗北。後発国家は建国直前の史実スナップショットから開始する。
- 長期キャンペーンは時代ごとに1ターンの年数を変え、全体を約300～400ターン・10～20時間に収める。
- 勝利は存続・軍事・科学・文化・経済。軍事等は即時勝利、存続は最終ターン判定。
- ウルク編の主軸は灌漑・人口・食料・都市周辺タイルの管理。運河は河川へ接続する1ヘックス施設で、隣接農地を灌漑するが維持と浚渫を必要とする。
- 人口は農民・牧畜民・漁民・職人・神官・兵士・労働者という役割と、自由・依存・奴隷化という地位を別軸で管理する。紀元前4000年開始時に奴隷人口を置かず、後代に導入する場合も確度を明示する。
- 経済は大麦、羊毛、葦、粘土、木材、石材、銅など実在物資を在庫・消費・交易へ接続する。
- 学術的推定を使う場合は「確実・有力説・推定」を明示し、図鑑の各項目に出典・年代幅・異説を表示する。
- 画像は博物館等の利用可能な資料を優先。不足時の生成画像は「復元イメージ」と権利条件を明記する。
- 音楽は確認可能な楽器を参考にした現代制作であり、当時の旋律そのものとは表示しない。
- 固有名詞は日本語通用名・原語・学術転写・異綴りへ対応する。
- Windows PC専用。日本語表示のみだが、ローカライズキーを全データに持たせる。
- 初心者向けに短い導入チュートリアルと任意の顧問を用意し、人口配置・生産・交易を個別に自動化できる。
- UIは簡潔／詳細切替。一般的な内蔵GPUで1080p・30fps、通常のAI待ち時間5秒以内を目標。
- 美術は歴史復元を意識したローポリ3D。移動・建設・収穫・洪水・戦闘・都市成長など情報を伝えるアニメーションを優先。
- 歴史要素はJSON／CSV中心のデータ駆動方式とし、開発時生成・史実確認・権利確認を通過した素材だけを収録する。

## 追加の `/grill-me` で確定した設計規則

- 史実イベントは年代で強制せず、年代窓・地理・資源・人口・制度を満たしたときに候補となる条件駆動型。条件不足なら遅延・不発・別展開になる。
- 証拠は `confirmed`（確実）、`probable`（有力説）、`inferred`（推定）に分け、初期配置・効果の強さ・表示にも反映する。
- 年表は出来事を「史実整合」「妥当な分岐」「仮想的分岐」に分類し、理由と根拠を表示する。
- 内部年代は正確なターン値で計算するが、画面は史料に応じて単年・年代幅・「頃」「推定」を使い分ける。
- 固定マップは確認・有力説・推定・ゲーム調整の復元根拠レイヤーを持つ。
- 難易度で初期資源や史実配置を変えない。顧問、警告、取り消し、AIの先読み、自動管理の設定で難度を調整し、AIにも霧と情報制約を適用する。
- 導入でも失敗を無効化しない。敗北時は直前自動セーブ、導入開始、助言強化、終了を選べる。
- 都市国家成立は、徴収・備蓄・再分配、記録行政、常設役職、民兵、防衛、正式な貢納・境界協定、専門区画を解禁する。同時に反発・労働負担・権力争いも増える。
- 食料には容量と保存損失があり、神殿倉庫・行政で改善し、洪水・不安・略奪で悪化する。
- 労働力は農業、運河、建設、工芸、交易、民兵が奪い合う。無理な動員は疲弊・逃亡・不安を招く。
- 役割と地位は分離し、依存、債務拘束、捕虜、奴隷化を同一視しない。史料不足の人数は推定幅として示す。
- 共同体・文化の系譜と政体を分離する。政体は集落連合・神殿共同体・都市国家・従属地域へ変化し、分裂・統合・消滅後も住民の系譜は残る。
- 近代的な国旗と固定国境は用いず、復元アイコン、居住圏、耕作圏、交易影響圏、利用権、貢納関係を表示する。
- 紛争は水利・土地争い、補償交渉、小規模略奪、人質・婚姻・共同祭祀、組織戦、正式戦争へ段階化する。
- 部隊は実人数、装備、士気、経験、食料を持つ集団。動員・死傷・捕虜・逃亡・帰還は人口と労働力へ反映する。
- 技術・制度は物資、実践、組織、担当者、最速年代などの知識要素で成立し、交易・移住・婚姻・捕虜・職人交流でも伝播する。
- 即時勝利後は終了またはターン50まで継続でき、最初の勝利種類と達成ターンを保持する。
- セーブは形式・キャンペーン・史実データセットの版数、シード、確度設定、安定ID、勝利履歴を持つ。確率処理はシード・ターン・イベントIDから再現可能にする。
- 重要処理は追記型イベント履歴へ記録し、プレイヤー向け年表と開発者向け計算ログを同じ事実から生成する。
- UIは用語の簡潔／詳細説明、文字サイズ、色以外の識別、線幅、動きの軽減、音量分離、キーボード操作、低解像度スクロールへ対応する。
- 通常30fps以上、入力反応100ms未満を性能目標とし、ローポリ、段階表示、画面外停止、プール、ターン集約計算、軽量モードで管理する。
- 音響は考古学的根拠・有力な復元・現代的補完・操作演出を区別する。画像・音響・模型も実物資料、学術復元、生成復元、ゲーム模型を混同しない。
- 生成素材は隔離生成し、モデル・設定・日時・プロンプトを記録して史実・権利確認を行い、承認済みだけをUnityへ収録する。
- ウルク版の収録範囲は紀元前4000～3000年頃の南メソポタミアと直接交流し得る要素に限る。後代・遠隔地は共通候補台帳へ分離する。

## 第1マイルストーンの実装

- `Assets/Resources/Campaigns/uruk_4000.json`
  - schema version 1、安定ID、ローカライズキー、日本語文、史料確度、出典参照。
  - 32×20の固定南メソポタミア地図、推定した下流河道、湿地側へ広がる水域。
  - ウルク、エリドゥ、ウル、ラガシュ地域、ニップール、キシュ地域、スシアナ交易圏、南部湿地共同体の8勢力。
  - 大麦、エンマー小麦、羊・羊毛、葦、沖積粘土、木材、石材、銅、魚の9物資。
- `Core/HistoricalCampaign.cs`
  - JSONデータ契約、ローカライズ、出典、確度、年代、勢力、物資、河川の検証。
  - 50ターンを紀元前4000年から前3000年へ割り当てる年代変換。将来は複数の可変年代区間を持てる。
- `Core/HistoricalCampaignFactory.cs`
  - 手続きマップを使わず固定JSONから既存Core互換の8勢力GameStateを構築。
  - 個人名が不明な指導者を捏造せず、「氏名不詳の祭司王」等の役職名で表示。
- `Core/HistoricalCampaignSave.cs`
  - `mode=historical_campaign` とcampaign IDを持つ外側エンベロープで、既存ランダムゲームのセーブと区別。
  - 内側のGameStateは既存SaveLoadへ委譲し、既存の移行処理を再利用。

## 第2マイルストーンの実装

- タイトル画面に「史実キャンペーン：ウルク」を追加し、既存ランダムゲームを残したまま別モードで開始できる。
- 小集落1つ、実人口1,500人、農地2区画、状態25%の未整備運河、わずかな食料備蓄から開始する。
- 役割別人口と地位別人口を同時に保持し、開始時の奴隷化人口は0人。人口配置だけ自動化ON、生産・交易はOFF。
- 大麦、エンマー小麦、羊・羊毛、葦、粘土、木材、石材、銅、魚を数量付きで保持する。
- 最初の3ターンは灌漑整備、食料優先、神殿計画を顧問が案内する。詳細表示では人口・地位・物資・自動化状態・復元注記を確認できる。
- 20年を1政策期間として集計し、最初の3回は固定、その後はシード決定の洪水傾向を適用する。
- 人口2,500人、灌漑農地2区画、食料安全、神殿完成、記録行政採用を都市国家成立条件として判定する。
- 軍事・科学・文化・経済の即時勝利、ターン50の存続勝利、中心集落占領・人口500人未満・重大不安3期の敗北条件を独立判定する。
- 史実専用セーブを通常の3スロットへ保存でき、3世代の自動セーブも保持する。データセット版数をセーブへ記録し、旧第1段階セーブは安全に補完する。
- 農地、運河、建設中の神殿を外部素材なしの暫定ローポリ復元模型として地図へ表示する。
- `Core/HistoricalContentSchema.cs` に共通台帳契約を追加。日本語名・原語・転写・異綴り、年代幅、地域、確度、異説、出典、関連ID、固有効果と社会的費用、素材来歴をJSON化できる。
- 製品収録ゲートは `verified` のみ許可する。参照日・資料種別・利用区分・ライセンス、生成復元のモデル・プロンプトハッシュ・確認者・「復元」表示を自動検査する。

## 地理表現上の注意

固定マップは現代の河道・海岸線をそのまま描いたものではない。UNESCO資料は紀元前5000～3000年頃に海水面が現在より約200km内陸へ達し、ウルク・ウル・エリドゥが淡水湿地の縁辺に位置したことを説明している。一方、個々の河道・政治境界を紀元前4000年の一点へ確定することはできない。このため地図全体を `inferred`、主要遺跡の存在を `probable` または `confirmed` として分離した。
