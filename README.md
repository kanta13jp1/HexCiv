# HexCiv — Civilization風 4Xストラテジー (Unity)

Unity 6.3 LTS (6000.3.20f1) 製の、シヴィライゼーション風ターン制ストラテジーゲームです。
外部アセット不要・スクリプトのみで動作します(マップ・ユニット・UIはすべて実行時に生成)。

## 遊び方(起動方法)

### A. Unityエディタで開く
1. Unity Hub で「開く」→ このフォルダ (`C:\Users\kanta\GitHub\HexCiv`) を選択(Unity 6.3 LTS)
2. `Assets/Scenes/Main.unity` を開いて ▶ Play
   (どのシーンでも Play すれば自動的にゲームが起動します)

### B. ビルド済みEXEで遊ぶ
`Build\HexCiv.exe` をダブルクリック(ビルド済みの場合)

起動すると**タイトル画面**が表示されます — 「つづける(クイックロード)」「新しいゲーム / 設定」「シミュレーション観戦」「そのまま遊ぶ」「終了」から選べます(Escで閉じてそのまま遊ぶことも可能)。
初回起動時はさらに6ページの「はじめてガイド」が表示されます。閉じた後も、画面右下の「？ 遊び方」からいつでも再表示できます。

## ゲーム概要

- **文明**: 世界各地域・各時代の92文明から選択可能（AI文明は重複なしで割当）
- **指導者**: 179件から文明ごとの指導者を選択可能（史料上の個人名が不明な場合は明記）
- **世界史図鑑**: 文明・指導者・演劇・映画を含む13分類、台帳1123件を6地域別に閲覧可能
- **遺産・偉人・作品**: 遺産を探索し、世界で一度だけ偉人を登用・作品を収蔵
- **国家運営**: 国庫、税制、都市・軍事維持費、安定度、戦争疲弊を管理（AIも自動判断）
- **マップ**: 毎回ランダム生成されるヘクスマップ(大陸・丘陵・森林・資源)
- **勝利条件**: 制覇、文化的影響を世界へ広げる文化勝利、または250ターン終了時のスコア勝利

### 基本ルール
- 開拓者で都市を建設 → 都市が食料で成長し、ユニットや建物を生産
- 技術を研究して新ユニット(弓兵・槍兵・剣士・カタパルト)や建物を解禁
- 1タイル1ユニット制。敵のいるタイルに移動すると攻撃(白兵)。弓兵・カタパルトは射程2の遠隔攻撃
- 丘陵・森林は防御ボーナス。都市はHP制で、白兵ユニットでとどめを刺すと占領
- 視界の外は戦場の霧(未探索は暗黒、探索済みは薄暗い)

- ユニット選択中は **黄色のマスが移動可能**、**赤いマスが攻撃可能**、白枠が現在地です
- 各ユニットの下に「開拓者」「戦士」などの名前が表示されます
- ユニットはタイル間を滑らかに移動し、戦闘時は突進モーション・被弾フラッシュ・**ダメージ数字**・撃破フェードの演出が入ります(8倍速以上の観戦では自動で簡略化)
- 海・沿岸タイルはゆっくり明滅する水面アニメーション付きです。待機中のユニットはゆるやかに上下し、防御態勢中はリングが明滅します
- **丘陵は立体的に盛り上がり**、ユニットや都市はその上に立ちます。領土内のタイルはうっすら文明色に染まります
- 資源はひと目で分かる形状アイコン(小麦=穂、牛・鹿=角、鉄=十字、馬=蹄鉄)で表示されます
- **都市は人口とともに街並みが成長**し、城壁を建設すると実際に石壁リングが出現します
- マップ上を**雲の影**がゆっくり流れ、画面全体の色調が時間とともにほんのり移ろいます。都市が陥落した瞬間はカメラが小さく揺れます

- 開拓者を選択して「都市建設」ボタンで都市を作ります(都市間は3タイル以上離す必要あり)
- 上部バーの「研究を選択」から技術を選びます
- 都市をクリックすると生産する物を選べます

## 文明選択

- 上部バーの「文明変更」から、地域・時代・固有色・固有都市名を持つ92文明を選べます
- 文明を選択すると、現在のゲームを終了して新しいマップで開始します
- 選択した文明は次回起動時にも保持されます
- 収録済み文明と今後の追加候補は `CIVILIZATIONS.md` を参照してください

## 指導者選択

- 上部バーの「指導者変更」から、現在の文明に属する指導者を選べます
- 指導者を選択すると新しいマップで開始し、選択は次回起動時とセーブデータに保持されます
- 収録済み179件、史料上の扱い、追加順は `LEADERS.md` を参照してください

## ゲーム設定

- 上部バーの「ゲーム設定」から、マップサイズ(小40x24/標準44x26/大56x34)・**マップ種別(大陸/パンゲア/群島)**・文明数(2〜8)・**難易度(やさしい/普通/むずかしい)**・シード(空欄=ランダム)を指定して新規ゲームを開始できます
- 難易度はAI文明の生産・科学・戦闘力を補正します(やさしい=AI 85%/戦闘-10%、むずかしい=AI 120%/戦闘+10%)
- 「フルスクリーン: ON/OFF」トグルで画面モードを切替できます(F11キーでも可。設定は次回起動時にも適用)
- 「演出: 標準/軽量」トグルで、雲影・水面ゆらぎ・待機アニメ・ダメージ数字を一括OFFにできます(低スペックPC向け)
- 設定は保存され、「文明変更」「指導者変更」からの新規ゲームにも適用されます
- 通常プレイ中も、宣戦布告・和平・都市陥落・文明滅亡などの重要イベントは画面上部に小さめのバナーで通知されます

## シミュレーションモード(観戦)

- ゲーム設定パネルの「**シミュレーション観戦で開始**」で、自分の文明もAIが操作する自動対戦が始まります
- **既定2倍速**で自動的にターンが進み、制覇または250ターンのゲーム終了まで実行されます
- 画面上部の観戦バーで「⏸ 一時停止 / ▶ 再開」「速度切替(等速〜**256倍速**の9段階)」ができます(高速時は1フレームに複数ターンを処理し、重要イベントの瞬間は自動で演出を優先)
- ゲーム終了画面には**全文明の最終スコア一覧**が表示され、「**最終戦況**」ボタンでスコア推移グラフも振り返れます。勝利時は紙吹雪とファンファーレ付きです
- **戦史年表**(Cキー/「年表」ボタン): 宣戦・和平・都市陥落・文明滅亡・勝敗をターン番号付きで自動記録。ゲーム終了後に世界の歴史を最初から振り返れます
- **歴史ツアー**(年表の「歴史ツアー」ボタン/終了画面): 記録された歴史イベントの現場をカメラが時系列で巡り、ダイジェスト映像のように歴史をたどれます(Escで中断)
- 占領された都市は**煙と残り火を上げ、数ターンかけて鎮火**します。建物が完成した都市には金色のスパークルが舞います
- 戦場の霧なしで全文明の拡張・戦争・和平を観戦できます。「観戦を終了して新規ゲーム」で通常プレイに戻ります
- **宣戦布告・都市陥落・文明滅亡・勝敗決定の瞬間はカメラが現場へジャンプ**し、バナー(「⚔ 宣戦布告!」「🏰 陥落」など)が表示されます
- 「**戦況**」ボタンで全文明の推移グラフを表示できます — タブで**スコア/軍事力/文化/技術**の4指標を切替(通常プレイ中も閲覧可)
- 年表の「**書き出し**」ボタンで、世界の歴史全記録をテキストファイルに保存できます(`chronicles/`フォルダ)
- 長期化した戦争は自動和平で終結することがあります(通常プレイでも同様)

## 世界史図鑑

- 左下の「世界史図鑑」から「総合」「文明」「指導者」「遺跡」「偉人」「研究」「文化」「作品」を切り替えて閲覧できます
- 総合索引には13分類・台帳1123件の現在値を表示します。地域ボタンを切り替えると、文明・指導者を含む全分類の地域別件数へ更新されます
- 図鑑上部には、既存作品や人物肖像を複製しないオリジナルの世界史パノラマ画像を表示します。文明・指導者・遺跡・偉人・研究・文化・作品7分類の各行には専用アイコンも表示します
- 地域ボタンで全件／アフリカ／西・南アジア／東・東南アジア／ヨーロッパ・地中海／アメリカ大陸／オセアニアを切り替えます
- 研究史132件は拡張技術ツリー、文化史132件は文化政策として実際のゲーム進行へ接続されています
- 標準マップには6地域から計12件の遺産が配置されます。探索済みタイルの金色「史跡」マーカーへユニットを移動すると、文化・科学・偉人ポイントを一度だけ獲得します
- 関連文明は遺産報酬+50%、同地域の文明は+25%。指導者は所属文明の地域親和性を受け継ぎます
- 都市・文化政策から毎ターン偉人ポイントを獲得します。左下「遺産・偉人」で、世界共通の132人から候補を選んで登用できます
- 偉人は活動分野に応じて、学術・文化・技術・社会・探検・軍事の6系統の効果を発動します。AI文明も自動登用します
- 作品史336件は書籍・絵画・彫刻・建築・音楽・演劇・映画が各48件、6地域が各56件です。作品ポイントで世界に一つだけ収蔵でき、AI文明も自動収蔵します
- 作品7分類の行には画像生成による専用アイコンを表示します。実在作品、ポスター、舞台写真は複製していません
- 全収録一覧と追加計画は `WORLD_HISTORY_CATALOG.md` を参照してください
- 研究史・文化史の全240件と収録基準は `RESEARCH_CULTURE_CATALOG.md` を参照してください
- 作品史の収録範囲・基準・追加手順は `MASTERPIECE_CATALOG.md` を参照してください
- 全分類の現在件数、共通基準、次バッチの順序は `GLOBAL_HISTORY_CATALOG.md` を参照してください

## セーブ／ロード

- 上部バーの「セーブ」「ロード」ボタンでスロット選択画面が開きます(3枠、各スロットにターン数・文明・保存日時と**その世界のミニマップサムネイル**を表示)
- **F5** はスロット1へのクイックセーブ、**F9** はスロット1からのクイックロード(自分の手番中のみ保存可)
- 保存先は `%USERPROFILE%\AppData\LocalLow\DefaultCompany\HexCiv\hexciv_save_slot1.json` ～ `slot3.json`(旧`hexciv_save.json`は初回起動時にスロット1へ自動移行)
- ターン開始時は未行動ユニットが自動選択されます
- 「はじめてガイド」は初回起動時のみ自動表示されます(以後は右下の「？ 遊び方」から表示)。ガイド中に説明どおりの操作をすると「✓ できました!」と表示されて自動で次のページへ進みます

## 文化と政策

- 都市、人口、首都、記念碑から毎ターン文化ポイントを獲得します
- 左下の「文化・政策」を開き、6地域の文化史から次に進める政策を選択します。政策は地域ごとに18段階です
- 政策を採用すると、文化・科学・都市生産・文化的影響力のいずれかが恒久的に強化されます
- 平和な文明同士は毎ターン文化交流を行い、相手文明への影響力が増えます。戦争中はその相手との交流が停止します
- 150ターン以降、政策14件・累計文化1500を満たし、全生存文明より文化が多く、全文明へ必要量の影響を与えると文化勝利です
- AI文明は文化政策を自動で選択します。採用政策と文明別影響力はセーブデータにも保存されます

## 世界遺産・偉人・作品収蔵

- 遺産の配置はゲームseedから決まり、同じ設定なら同じ場所に出現します。開始地点から到達可能な陸地へ配置されます
- 未発見遺産は、タイルを探索すると地図に金色の「史跡」として現れます。ユニットをそのタイルへ移動して発見します
- 発見報酬はゲーム進行に合わせて増え、関連文明・同地域文明は追加報酬を得ます
- 偉人は世界全体で一度だけ登用できます。関連文明・同地域文明は必要ポイントが20%／10%軽減されます
- 都市・文化政策・登用済み偉人から作品ポイントを得ます。「作品収蔵」タブで336件から選び、書籍は科学、絵画・音楽は文化的影響力、彫刻・建築は都市生産、演劇・映画は複合効果を発動します
- 作品は世界全体で一度だけ収蔵できます。関連文明・同地域・関連偉人によって費用が軽減され、北斎と「神奈川沖浪裏」のような直接関係では偉人登用時に作品が生まれます
- 遺産、偉人、作品ポイント、発見・登用・収蔵状況、国庫・税制・安定度・戦争疲弊はセーブデータ（version 10）へ保存されます

## 国家運営

- 人口・都市・建物から税収を得て、都市・ユニット・戦争の維持費を支払います
- 減税は安定度と科学・文化・生産を支え、重税は税収を増やす代わりに安定度と産出へ負担をかけます
- 国庫赤字、首都喪失、長期戦による戦争疲弊は安定度を下げます
- AI文明も国庫、戦争、安定度を見て税制を自動変更します
- シミュレーションゲーム要素と画像・動画・音楽・音声生成技術の段階導入計画は `SIMULATION_AND_GENERATION_CATALOG.md` を参照してください

## サウンド

- 落ち着いた戦略BGMを収録。**ターン100・180を境に3段階のバリエーションへクロスフェード**し、時代の進行を感じられます(トップバーの**時代表示**「古代/中世/近代」と同期)
- タイトル画面では専用の静かなイントロ曲が流れ、ゲーム開始で通常BGMへクロスフェードします
- **選んだ文明の地域に合わせてBGMの旋律が変化**します(東・東南アジア=五音音階、西・南アジア=ハーモニックマイナー、アメリカ大陸=開いた和音など6種)
- ボタン、ユニット選択、移動、戦闘、都市建設、ターン終了、研究・生産完了、勝敗にSEを追加
- **戦闘音はユニット種別で変化**します(白兵=金属の剣戟、弓兵=矢の風切り音、カタパルト=重い着弾音)
- **遺産発見・偉人登用・作品収蔵**にも専用のチャイム音、**宣戦布告・都市陥落**には専用のスティング音があります
- BGMの下には静かな風の環境音レイヤーが流れます(BGM音量・ミュートに連動)
- 初期音量はBGM 35%、SE 70%。設定は次回起動時にも引き継がれます
- 音はUnity実行時に独自生成するため、外部音源ファイルや追加ライセンスは不要です

## 技術ツリー

前半は従来の12基礎技術です。

```
農業 ─┬─ 陶器 ── 筆記(図書館)
      ├─ 畜産 ── 車輪 ── 数学(カタパルト)
      ├─ 弓術(弓兵)
      └─ 採鉱 ─┬─ 青銅器(槍兵)── 鉄器(剣士)
               └─ 石工術(城壁)── 建築学(作業場)
```

12基礎技術をすべて習得すると、世界史図鑑の研究史132件が実際の研究対象として開きます。

- アフリカ／西・南アジア／東・東南アジア／ヨーロッパ・地中海／アメリカ大陸／オセアニアの6分岐
- 各地域22段階。旧セーブ互換のため既存20件の順序を維持し、第4弾2件を後段へ追加
- 第1段階コスト110、以降は1段階ごとに+35
- 研究内容・時期・分野は左下「世界史図鑑」→「研究史」でも確認可能
