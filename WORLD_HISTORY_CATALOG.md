# 世界史図鑑：遺跡・史跡と偉人

## 収録方針

「地球上・歴史上のすべての遺跡」と「すべての偉人」には、有限の確定リストがありません。
遺跡は未発見・未発掘の場所が増え続け、考古遺跡、現存建築、文化的景観、記憶の場所の
境界も一意ではありません。偉人は価値判断を含むため、本作では人物の優劣を順位付けせず、
故人の歴史的活動を分野・地域横断で記録します。

UNESCOの文化遺産定義が含む「記念物・建造物群・考古遺跡を含む場所」を参考に、ゲーム内の
「遺跡・史跡」は考古遺跡だけでなく歴史的建造物と文化的景観を含みます。世界遺産への登録は
収録条件でも価値順位でもありません。第3弾までに
**遺跡・史跡108件＋偉人120人＝228件** を実装しています。

## ゲーム内実装

- 左下の「世界史図鑑」ボタンからいつでも開ける
- 「遺跡・史跡」「偉人」のタブ切替
- 6地域と全件表示をボタンで循環
- 6件ずつのページ表示。名称、場所・時代、種別・分野、短い説明を表示
- 総合分類と各項目へ、遺跡／偉人を表すオリジナル2分割エンブレムを表示
- `UIManager` と独立したCanvasのため、文明選択・指導者選択・ゲーム設定と競合しない

## ゲームへの接続（実装済み）

1. 標準マップへ6地域×2件、計12件をseed由来で決定的に配置。開始文明から到達可能な陸地を優先
2. 探索済みタイルへ金色の史跡マーカーを表示し、ユニット到達時に文化・科学・偉人ポイントを一度だけ付与
3. 関連文明は報酬+50%、同地域文明は+25%。所属指導者も文明の地域親和性を受け継ぐ
4. 都市数と文化政策から毎ターン偉人ポイントを獲得。偉人120人を世界共通プールから一度だけ登用
5. 活動分野を学術・文化・技術・社会・探検・軍事の6系統へ分類し、登用時に固有の即時効果を発動
6. 独立「遺産・偉人」画面、世界史図鑑の発見／登用状態、AI自動登用、セーブversion 9へ対応

## 順次実装する次段階

1. 遺産の種別ごとの個別イベントと選択式報酬
2. 偉人の年代記、複数回使用型能力、都市への作品・学術成果の収蔵
3. 追加文明と結び付く遺跡・人物を地域単位で増補
4. Wikidata ID、UNESCO資産ID、典拠URL、異名、座標、年代範囲を台帳へ追加
